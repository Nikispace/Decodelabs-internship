1. Overview

   Python projects built during my DecodeLabs journey, coded and tested in Python IDLE and versioned with Git and GitHub.

2. Tools

   Python 3.x  
   Python IDLE  
   Git and GitHub  

3. How to run

   Open the .py file in IDLE  
   Press F5 to run  
   Install extra libraries in Command Prompt using  

   python -m pip install package_name  

4. Workflow

   Edit in IDLE  
   Test in IDLE  
   git add .  
   git commit -m "message"  
   git push  

5. Purpose

   Practise core Python, build small working scripts, and grow a clean, readable project portfolio.
